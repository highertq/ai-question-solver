# 识题王 - AI智能解题助手

一款基于视觉AI的智能解题助手，通过摄像头拍摄作业，使用OK手势拍照上传题目后自动获取AI解答。

## 功能特性

### 核心功能

| 功能 | 描述 |
|------|------|
| 摄像头实时预览 | 支持调用设备摄像头，实时显示作业画面 |
| 鼠标框选 | 使用鼠标拖拽框选试卷上的题目区域 |
| OK手势拍照 | 比OK手势保持1秒自动拍照上传题目 |
| AI智能解题 | 自动识别题目图片，给出详细解答步骤 |
| 追问对话 | 在AI解答页面可以继续追问，多轮对话 |
| AI智能写作 | 支持多种文章类型、文体和风格，智能生成文章 |
| AI知识点讲解 | 输入任意知识点，AI详细讲解概念、要点和应用 |
| AI内容审核 | 检测文本违规内容、敏感词、垃圾信息和内容质量 |
| AI百科全书 | 输入关键词，AI提供百科知识、历史背景和应用 |
| AI程序助手 | 支持14种编程语言，提供代码调试、优化、生成等功能 |
| 计算器 | 支持基本运算、三角函数和方程求解的纯本地计算器 |
| 积分系统 | 使用AI功能获得积分，积分可用于玩游戏 |
| 小恐龙快跳 | 消耗20积分玩一次，得分≥100奖励10积分 |
| 迷宫探险 | 成功走出迷宫奖励30积分 |
| 每日登录奖励 | 每日登录可获得100积分奖励 |
| 错题本 | 本地存储错题及解答，支持查看和删除 |
| 流式输出 | AI回答实时显示，提升阅读体验 |

### 工具栏功能

- **摄像头** - 启动/关闭摄像头
- **框选** - 进入框选模式（鼠标拖拽框选）
- **错题本** - 查看历史保存的错题
- **AI解题** - 查看当前题目的AI解答
- **AI写作** - 打开AI智能写作侧边栏，生成各类文章
- **AI知识点** - 打开AI知识点讲解侧边栏，学习各类知识点
- **AI审核** - 打开AI内容审核侧边栏，检测违规内容
- **AI百科** - 打开AI百科全书侧边栏，查询百科知识
- **AI编程** - 打开AI程序助手侧边栏，解决编程问题
- **游戏** - 打开小恐龙快跳游戏，消耗20积分
- **迷宫** - 打开迷宫探险游戏，成功奖励30积分

- **计算器** - 右下角可拖动悬浮球，点击打开计算器侧边栏

## 技术栈

- **前端**: 纯HTML/CSS/JavaScript（无框架依赖）
- **AI模型**: 阿里云千问 VL 多模态模型 (`qwen-vl-max`)
- **手势识别**: MediaPipe Hands
- **存储**: LocalStorage

## API 配置

### 使用的模型

| 模型 | 用途 | 说明 |
|------|------|------|
| `qwen3.5-plus` | 文本生成（首选） | 阿里云千问模型，用于追问对话 |
| `qwen-vl-max` | 图像理解（备用） | 阿里云千问视觉语言模型，支持图像输入 |

### API 端点

```
Base URL: https://dashscope.aliyuncs.com/compatible-mode/v1
Endpoint: POST /chat/completions
```

### 请求格式

```javascript
{
  "model": "qwen-vl-max",
  "messages": [
    {
      "role": "system",
      "content": "你是一位专业的教育辅导老师..."
    },
    {
      "role": "user",
      "content": [
        { "type": "text", "text": "题目描述..." },
        { "type": "image_url", "image_url": { "url": "data:image/png;base64,..." } }
      ]
    }
  ],
  "stream": true
}
```

## 快速开始

### 1. 打开应用

直接双击以下文件即可使用：
- **启动.html** - 启动页面（推荐）
- **index.html** - 主应用
- **question-bank.html** - 题库页面

### 2. 使用流程

#### AI解题流程

1. 点击左侧工具栏的「摄像头」按钮启动摄像头
2. 将摄像头对准作业本
3. **OK手势方式**: 比OK手势保持1秒自动拍照上传题目
4. **鼠标方式**: 点击「框选」按钮，拖拽鼠标框选题目
5. 系统自动分析题目并显示AI解答
6. 在底部对话框输入追问内容，按Enter发送
7. 点击「保存到错题本」保存题目

#### AI写作流程

1. 点击左侧工具栏的「AI写作」按钮打开侧边栏
2. 选择文章类型（议论文、记叙文、说明文等）
3. 设置目标字数、写作风格、文体和目标读者
4. 输入写作主题和具体要求
5. 点击「开始写作」生成文章
6. 可复制或保存生成的文章

#### AI知识点讲解流程

1. 点击左侧工具栏的「AI知识点」按钮打开侧边栏
2. 输入想要学习的知识点名称（如"二次函数"、"光合作用"等）
3. 点击「讲解」按钮
4. AI会详细讲解知识点的定义、要点、例子和应用
5. 可复制讲解内容保存

#### AI内容审核流程

1. 点击左侧工具栏的「AI审核」按钮打开侧边栏
2. 选择审核类型（文本内容、敏感词检测、垃圾信息检测、内容质量评估）
3. 输入需要审核的内容
4. 点击「开始审核」按钮
5. AI会提供审核结论、风险等级、发现的问题和修改建议
6. 可复制审核结果

#### AI百科全书流程

1. 点击左侧工具栏的「AI百科」按钮打开侧边栏
2. 输入想要查询的关键词（如"量子力学"、"文艺复兴"等）
3. 点击「查询」按钮
4. AI会提供准确的定义、历史背景、主要特点和实际应用
5. 可复制百科内容保存

#### AI程序助手流程

1. 点击左侧工具栏的「AI编程」按钮打开侧边栏
2. 选择编程语言（Python、JavaScript、Java等14种语言）
3. 选择任务类型（代码调试、代码优化、代码生成等）
4. 输入问题描述或粘贴代码
5. 点击「生成答案」按钮
6. AI会提供问题分析、解决方案、代码解释和注意事项
7. 可复制代码结果

#### 计算器使用流程

1. 点击右下角的可拖动悬浮球打开计算器侧边栏
2. 选择计算模式：
   - **基本运算**：支持加减乘除、括号、百分比等
   - **三角函数**：支持sin、cos、tan等三角函数
   - **方程求解**：支持一元一次方程和一元二次方程求解
3. 在输入框中输入表达式或方程
4. 点击「=」按钮或「求解」按钮获取结果
5. 计算器为纯本地离线功能，无需网络连接

## 游戏说明

### 小恐龙快跳

1. 点击左侧工具栏的「游戏」按钮打开侧边栏
2. 点击「开始游戏」按钮，消耗20积分
3. 按空格键或点击屏幕跳跃
4. 躲避红色障碍物，每通过一个障碍物得10分
5. 得分≥100时获得10积分奖励
6. 游戏速度会随着分数增加而加快

### 迷宫探险

1. 点击左侧工具栏的「迷宫」按钮打开侧边栏
2. 点击「开始游戏」按钮
3. 使用方向键或WASD移动青色方块
4. 从左上角起点移动到右下角黄色终点
5. 成功到达终点获得30积分奖励
6. 可以随时点击「重新开始」生成新迷宫

### 每日奖励

1. 点击左侧工具栏的「奖励」按钮打开侧边栏
2. 检查今日是否已领取奖励
3. 点击「领取每日奖励」按钮获得100积分
4. 每天只能领取一次，次日自动重置

## 手势操作说明

### OK手势拍照

| 手势 | 操作 |
|------|------|
| OK手势（拇指食指成圈，其他手指伸直） | 保持1秒自动拍照上传 |
| 其他手势 | 无操作 |

### 手势识别原理

1. 使用 MediaPipe Hands 实时检测手部关键点
2. 判断拇指和食指指尖是否形成圆圈（距离小于阈值）
3. 判断中指、无名指、小指是否伸直
4. 满足条件后开始计时，保持1秒触发拍照

## 注意事项

1. **摄像头权限**: 首次使用需要授权摄像头访问权限
2. **网络要求**: 需要联网访问阿里云API
3. **光线条件**: 确保作业区域光线充足，提高识别准确率
4. **手势识别**: 手势功能需要手部在摄像头画面中清晰可见

## API Key 安全建议

生产环境中，建议将 API Key 存储在后端服务中，而不是直接暴露在前端代码中。当前实现仅用于演示目的。
